Summary

This project models how autonomous agents can evolve advantageous traits across generations while learning within their lifetimes. It integrates Evolutionary Algorithms (EAs) and Deep Reinforcement Learning (RL) to simulate complex, adaptive behaviors in a multi-agent ecosystem. The simulation was ported from Unity to Python for performance and scalability, offering a testbed for artificial life and intelligent system studies.

Key Features

  • Hybrid Evolutionary AI + RL architecture
  • Dual-stream CNN + MLP sensory system
  • Emergent behaviors like kin cooperation and resource adaptation
  • Real-time simulation visualization

Results and Impact

The hybrid system outperforms standalone EA and RL models in fitness, adaptability, and behavioral complexity. The project offers a foundation for future research in artificial life, game AI, and education.
