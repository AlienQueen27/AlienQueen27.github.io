📌 Project Summary

This project models adaptive behavior in artificial agents through a hybrid system combining Evolutionary Algorithms (EA) and Reinforcement Learning (RL). Agents process environmental stimuli via CNNs and internal state via MLPs, learning both over their lifetimes and across generations.

🧠 Key Features

  • Hybrid Evolutionary AI + RL architecture
  • Dual-stream CNN + MLP sensory system
  • Emergent behaviors like kin cooperation and resource adaptation
  • Real-time simulation visualization

📊 Results

The hybrid approach achieved 90% higher fitness than EA-only and 67% higher than RL-only agents. It also demonstrated more stable policies and richer social dynamics. These findings show the power of combining evolutionary and reinforcement learning techniques in complex environments.
